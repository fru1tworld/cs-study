RFC 4632 (Classless Inter-domain Routing - CIDR):

서브넷 마스크, /24 같은 표기법, 그리고 IP 주소 고갈을 막기 위한 주소 할당 방식(CIDR)을 다룹니다.

3. 연결 및 전송 계층 (Transport Layer & Helpers)
데이터를 어떻게 신뢰성 있게(혹은 빠르게) 보낼 것인가에 대한 문서들입니다.

RFC 793 (Transmission Control Protocol - TCP):

TCP의 바이블입니다. 3-way handshake, 흐름 제어, 오류 제어의 기초가 담겨 있습니다. (※ 최신 통합본은 RFC 9293이지만, 역사적 맥락과 기초는 793이 유명합니다.)

RFC 768 (User Datagram Protocol - UDP):

매우 짧습니다. UDP가 얼마나 단순한 프로토콜인지 눈으로 직접 확인하기 좋습니다.
