RFC 826 (Ethernet Address Resolution Protocol - ARP):

IP 주소를 MAC 주소로 어떻게 변환하는지(L3와 L2의 연결) 설명합니다.

RFC 792 (Internet Control Message Protocol - ICMP):

ping이나 traceroute가 어떻게 작동하는지, 네트워크 오류 메시지는 어떻게 전달되는지 설명합니다.

4. 핵심 인프라 프로토콜 (Infrastructure)
네트워크를 편리하게 사용하게 해주는 도우미 프로토콜입니다.

RFC 1034 (Domain Names - Concepts and Facilities):

DNS의 기본 개념, 계층 구조, 네임 서버의 역할을 설명합니다.
