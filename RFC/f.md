RFC 3022 (Traditional IP Network Address Translator - NAT):

공유기 환경에서 필수적인 NAT(네트워크 주소 변환)의 동작 원리를 설명합니다.

5. 웹 및 응용 계층 (Application Layer - Web & Security)
개발자가 가장 자주 접하는 웹 관련 표준입니다.

RFC 9110 (HTTP Semantics):

기존의 복잡한 HTTP RFC들이 정리되어 나온 최신 HTTP 바이블입니다. HTTP 메서드(GET, POST), 상태 코드, 헤더의 의미를 다룹니다. (기존 RFC 2616, 7231 등을 대체)

RFC 9112 (HTTP/1.1):

HTTP/1.1의 메시지 문법과 파싱에 집중한 문서입니다.

RFC 7540 (Hypertext Transfer Protocol Version 2 - HTTP/2):

HTTP/1.1의 성능 문제를 해결하기 위한 멀티플렉싱, 헤더 압축(HPACK) 등을 다룹니다.

RFC 8446 (The Transport Layer Security (TLS) Protocol Version 1.3):

HTTPS의 핵심인 TLS의 최신 버전입니다. 핸드셰이크 과정이 어떻게 간소화되고 보안이 강화되었는지 보여줍니다.
