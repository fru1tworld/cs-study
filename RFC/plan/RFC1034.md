## 요약 형식 규칙

### 개조식 작성 원칙
도메인 구조는 트리 다이어그램으로 표현:
```
                 .
        /        |        \
      com       org       net
     /   \       |
   foo   bar   baz
```

### RR 타입 표
```markdown
| Type | 값 | 설명 |
|------|-----|------|
| A | 1 | IPv4 주소 |
```
