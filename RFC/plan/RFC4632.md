## 요약 형식 규칙

### 개조식 작성 원칙
```markdown
## 섹션 제목

### 하위 주제
- 핵심 포인트 1
  - 세부 사항 a
  - 세부 사항 b
- 핵심 포인트 2
```

### CIDR 표기법 예시
```markdown
| CIDR | 서브넷 마스크 | 호스트 수 |
|------|--------------|----------|
| /8 | 255.0.0.0 | 16,777,214 |
| /16 | 255.255.0.0 | 65,534 |
| /24 | 255.255.255.0 | 254 |
| /32 | 255.255.255.255 | 1 |
```

### 라우트 집계 다이어그램
```
192.168.0.0/24 \
192.168.1.0/24  |---> 192.168.0.0/22 (집계)
192.168.2.0/24  |
192.168.3.0/24 /
```
