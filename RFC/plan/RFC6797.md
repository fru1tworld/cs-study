## 요약 형식 규칙

### 개조식 작성 원칙
```markdown
## 섹션 제목

### 하위 주제
- 핵심 포인트 1
  - 세부 사항 a
  - 세부 사항 b
- 핵심 포인트 2
```

### STS 헤더 형식
```
Strict-Transport-Security: max-age=<seconds>; includeSubDomains
```

### HSTS 동작 흐름
```markdown
1. 클라이언트가 http://example.com 요청
2. HSTS 정책이 없으면 HTTP로 연결
3. 서버가 Strict-Transport-Security 헤더로 응답
4. 클라이언트가 HSTS 정책 저장
5. 이후 요청은 자동으로 https://로 변환
```
