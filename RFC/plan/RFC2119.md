# RFC 2119 요약 작업 지시서

## 대상 문서
- **RFC 번호**: 2119
- **제목**: Key words for use in RFCs to Indicate Requirement Levels
- **원본 URL**: https://www.rfc-editor.org/rfc/rfc2119

---

## 작업 목표
RFC 2119 원문을 **개조식으로 완전 요약**하되, **어떠한 내용도 누락하지 않아야 함**.

---

## 작업 지침

### 1. 원문 섹션별 완전 매핑
원본 RFC의 모든 섹션을 식별하고, 각 섹션의 **모든 내용**을 빠짐없이 포함해야 합니다.

### 2. 누락 방지 체크리스트

다음 항목들이 반드시 포함되어야 합니다:

#### 문서 메타정보

- [ ] 문서 목적 요약

- [ ] 정의: 절대적 요구사항

#### 2. MUST NOT

- [ ] 정의: 비권장사항 (예외 있을 수 있음)

## 요약 형식 규칙

### 개조식 작성 원칙
```markdown
## 섹션 제목

### 하위 주제
- 핵심 포인트 1
  - 세부 사항 a
  - 세부 사항 b
- 핵심 포인트 2
```

### 키워드 정리 표
```markdown
| 키워드 | 동의어 | 의미 |
|--------|--------|------|
| MUST | REQUIRED, SHALL | 절대적 요구사항 |
| MUST NOT | SHALL NOT | 절대적 금지사항 |
| SHOULD | RECOMMENDED | 권장사항 |
| SHOULD NOT | NOT RECOMMENDED | 비권장사항 |
| MAY | OPTIONAL | 선택적 사항 |
```
