```markdown
## 섹션 제목

### 하위 주제
- 핵심 포인트 1
  - 세부 사항 a
  - 세부 사항 b
- 핵심 포인트 2
```

### NAT 주소 변환 다이어그램
```
내부 네트워크          NAT 장치          외부 네트워크
[192.168.1.10] ---> [NAT: 203.0.113.1] ---> [Internet]
```

### 용어 매핑 표
```markdown
| 용어 | 설명 |
|------|------|
| Inside Local | 내부 네트워크에서 사용하는 내부 호스트 주소 |
| Inside Global | 외부에서 보이는 내부 호스트 주소 |
```
