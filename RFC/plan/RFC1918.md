# RFC 1918 요약 작업 지시서

## 대상 문서
- **RFC 번호**: 1918
- **제목**: Address Allocation for Private Internets
- **원본 URL**: https://www.rfc-editor.org/rfc/rfc1918

---

## 작업 목표
RFC 1918 원문을 **개조식으로 완전 요약**하되, **어떠한 내용도 누락하지 않아야 함**.

## 요약 형식 규칙

### 개조식 작성 원칙
```markdown
## 섹션 제목

### 하위 주제
- 핵심 포인트 1
  - 세부 사항 a
  - 세부 사항 b
- 핵심 포인트 2
```

### 사설 주소 범위 표
```markdown
| 범위 | CIDR | 클래스 | 주소 수 |
|------|------|--------|---------|
| 10.0.0.0 - 10.255.255.255 | 10/8 | A | 16,777,216 |
| 172.16.0.0 - 172.31.255.255 | 172.16/12 | B | 1,048,576 |
| 192.168.0.0 - 192.168.255.255 | 192.168/16 | C | 65,536 |
```
