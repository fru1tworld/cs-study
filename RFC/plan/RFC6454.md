## 요약 형식 규칙

### 개조식 작성 원칙
```markdown
## 섹션 제목

### 하위 주제
- 핵심 포인트 1
  - 세부 사항 a
  - 세부 사항 b
- 핵심 포인트 2
```

### Origin 구성 요소 표
```markdown
| 구성 요소 | 설명 | 예시 |
|----------|------|------|
| scheme | 프로토콜 | http, https |
| host | 호스트명 | example.com |
| port | 포트 번호 | 80, 443 |
```

### Origin 비교 예시
```markdown
| URL A | URL B | 동일 Origin? | 이유 |
|-------|-------|-------------|------|
| http://example.com/a | http://example.com/b | Yes | scheme, host, port 동일 |
| http://example.com | https://example.com | No | scheme 다름 |
```
