```markdown

  - 세부 사항 a
  - 세부 사항 b
- 핵심 포인트 2
```

### DHCP 교환 흐름 다이어그램
```
Client                Server
   |                    |
   |--DHCPDISCOVER-->   |
   |                    |
   |<---DHCPOFFER---    |
   |                    |
   |--DHCPREQUEST--->   |
   |                    |
   |<---DHCPACK-----    |
```

### 상태 전이 표
```markdown
| 현재 상태 | 이벤트 | 다음 상태 |
|-----------|--------|-----------|
| INIT | DHCPDISCOVER 전송 | SELECTING |
```
