## 대상 문서

- **제목**: An Ethernet Address Resolution Protocol (ARP)

## 작업 목표

## 요약 형식 규칙

### 개조식 작성 원칙
```markdown
## 섹션 제목

### 하위 주제
- 핵심 포인트 1
  - 세부 사항 a
  - 세부 사항 b
- 핵심 포인트 2
```

### 패킷 형식 다이어그램
반드시 포함:
```
+-------+-------+-------+-------+
|     Hardware Type             |
+-------+-------+-------+-------+
|     Protocol Type             |
...
```

### ARP 동작 흐름
요청-응답 시퀀스 다이어그램 포함
