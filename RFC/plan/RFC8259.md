## 요약 형식 규칙

### 개조식 작성 원칙
```markdown
## 섹션 제목

### 하위 주제
- 핵심 포인트 1
  - 세부 사항 a
  - 세부 사항 b
- 핵심 포인트 2
```

### JSON 값 유형 표
```markdown
| 값 유형 | 설명 | 예시 |
|---------|------|------|
| object | 순서 없는 이름/값 쌍 | {"name": "value"} |
| array | 순서 있는 값 목록 | [1, 2, 3] |
| number | 숫자 | 123, -45.67, 1e10 |
| string | 문자열 | "hello" |
| true | 불리언 참 | true |
| false | 불리언 거짓 | false |
| null | 널 값 | null |
```

### 이스케이프 시퀀스 표
```markdown
| 시퀀스 | 의미 |
|--------|------|
| \" | 큰따옴표 |
| \\ | 역슬래시 |
| \/ | 슬래시 |
| \b | 백스페이스 |
| \f | 폼 피드 |
| \n | 줄 바꿈 |
| \r | 캐리지 리턴 |
| \t | 탭 |
| \uXXXX | 유니코드 |
```
