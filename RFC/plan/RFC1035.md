## 검증 방법

### 완료 후 체크
1. 모든 RR 타입이 상세히 설명되었는지 확인

3. RCODE 값들이 모두 포함되었는지 확인
4. 메시지 압축 방식이 설명되었는지 확인
5. UDP/TCP 사용 조건이 설명되었는지 확인
