# EC2 인스턴스 타입

## 개요

EC2 인스턴스 타입은 CPU, 메모리, 스토리지, 네트워킹 용량의 다양한 조합을 제공합니다. 애플리케이션의 요구사항에 맞는 적절한 리소스 조합을 선택할 수 있습니다.

## 인스턴스 타입 명명 규칙

인스턴스 타입 이름은 다음과 같은 구조로 구성됩니다:

```
[패밀리][세대][추가기능].[크기]

예: m5.xlarge, c6g.2xlarge, r5dn.large
```

| 구성요소 | 설명 | 예시 |
|----------|------|------|
| **패밀리** | 인스턴스의 최적화 영역 | m(범용), c(컴퓨팅), r(메모리) |
| **세대** | 인스턴스 세대 번호 | 5, 6, 7 |
| **프로세서** | 프로세서 종류 | a(AMD), g(Graviton), i(Intel) |
| **추가기능** | 특수 기능 | n(네트워크), d(로컬 스토리지), e(확장 메모리) |
| **크기** | 인스턴스 크기 | nano, micro, small, medium, large, xlarge |

## 인스턴스 패밀리

### 범용(General Purpose)

균형 잡힌 컴퓨팅, 메모리, 네트워킹 리소스를 제공합니다.

| 패밀리 | 프로세서 | 사용 사례 |
|--------|----------|-----------|
| **M5** | Intel Xeon | 웹 서버, 개발/테스트 환경, 소규모 DB |
| **M6i** | Intel Xeon 3세대 | 범용 워크로드, SAP 애플리케이션 |
| **M6g** | AWS Graviton2 | 비용 효율적인 범용 워크로드 |
| **M7g** | AWS Graviton3 | 최신 범용 워크로드 |
| **T3** | Intel Xeon | 버스트 성능이 필요한 가변 워크로드 |
| **T4g** | AWS Graviton2 | 비용 효율적인 버스트 워크로드 |

### 컴퓨팅 최적화(Compute Optimized)

고성능 프로세서를 사용하는 컴퓨팅 집약적 작업에 적합합니다.

| 패밀리 | 프로세서 | 사용 사례 |
|--------|----------|-----------|
| **C5** | Intel Xeon | 배치 처리, 게임 서버, 과학 모델링 |
| **C6i** | Intel Xeon 3세대 | 고성능 컴퓨팅 |
| **C6g** | AWS Graviton2 | 비용 효율적인 고성능 컴퓨팅 |
| **C7g** | AWS Graviton3 | 최신 컴퓨팅 최적화 워크로드 |

### 메모리 최적화(Memory Optimized)

대용량 데이터셋을 메모리에서 처리하는 워크로드에 적합합니다.

| 패밀리 | 프로세서 | 사용 사례 |
|--------|----------|-----------|
| **R5** | Intel Xeon | 고성능 데이터베이스, 인메모리 캐시 |
| **R6i** | Intel Xeon 3세대 | SAP HANA, 실시간 분석 |
| **R6g** | AWS Graviton2 | 오픈소스 DB, 인메모리 캐싱 |
| **X2idn** | Intel Xeon | 대규모 인메모리 DB |
| **U-6tb1** | Intel Xeon | SAP HANA (6TB 메모리) |

### 스토리지 최적화(Storage Optimized)

대용량 데이터셋에 대한 높은 순차 읽기/쓰기 액세스가 필요한 워크로드에 적합합니다.

| 패밀리 | 특징 | 사용 사례 |
|--------|------|-----------|
| **I3** | NVMe SSD | 고성능 트랜잭션 DB |
| **I4i** | AWS Nitro SSD | NoSQL DB, 데이터 웨어하우징 |
| **D3** | HDD | 분산 파일 시스템, 데이터 레이크 |
| **H1** | HDD | MapReduce, 분산 파일 시스템 |

### 가속 컴퓨팅(Accelerated Computing)

하드웨어 가속기 또는 코프로세서를 사용합니다.

| 패밀리 | 가속기 | 사용 사례 |
|--------|--------|-----------|
| **P5** | NVIDIA H100 | 대규모 ML 학습 |
| **P4** | NVIDIA A100 | ML 학습, HPC |
| **G5** | NVIDIA A10G | 그래픽 집약적 애플리케이션, ML 추론 |
| **Inf2** | AWS Inferentia2 | ML 추론 |
| **Trn1** | AWS Trainium | ML 학습 |
| **DL1** | Habana Gaudi | 딥러닝 학습 |

### 고성능 컴퓨팅(HPC)

고성능 컴퓨팅 워크로드에 최적화되어 있습니다.

| 패밀리 | 특징 | 사용 사례 |
|--------|------|-----------|
| **Hpc6a** | AMD EPYC | 대규모 시뮬레이션 |
| **Hpc7g** | AWS Graviton3E | CFD, 날씨 모델링 |

## 프로세서 옵션

### Intel Xeon
- AES-NI 암호화 지원
- AVX, AVX2, AVX-512 명령어 지원
- Intel Turbo Boost 기술

### AMD EPYC
- 비용 효율적인 대안
- 인스턴스 타입에 'a' 접미사 (예: m5a)

### AWS Graviton
- ARM 기반 프로세서
- 최대 40% 향상된 가격 대비 성능
- 인스턴스 타입에 'g' 접미사 (예: m6g)

### AWS Trainium
- ML 학습 전용 가속기
- 대규모 딥러닝 모델 학습에 최적화

### AWS Inferentia
- ML 추론 전용 가속기
- 비용 효율적인 추론 워크로드

## 하이퍼바이저

### AWS Nitro System
- 최신 세대 인스턴스의 기반
- 향상된 보안 및 성능
- 베어메탈과 유사한 성능 제공

### Xen
- 이전 세대 인스턴스에서 사용
- HVM(Hardware Virtual Machine) 가상화
