# IAM 고급 기능

IAM의 고급 기능과 엔터프라이즈 환경에서의 활용 방법을 설명합니다.

## 목차

1. [IAM Access Analyzer](./access-analyzer.md)
   - 외부 접근 분석
   - 미사용 접근 분석
   - 정책 검증
   - 정책 생성

2. [교차 계정 접근](./cross-account-access.md)
   - IAM 역할 기반 접근
   - 외부 ID 사용
   - 리소스 기반 정책
   - 역할 체이닝

3. [Organizations SCP](./organizations-scp.md)
   - 서비스 제어 정책 개요
   - SCP 전략 (거부 목록 vs 허용 목록)
   - 일반적인 SCP 예시
   - SCP 관리

## 고급 기능 개요

### IAM Access Analyzer

```
주요 기능:
┌─────────────────────────────────────────────────────────────┐
│                    IAM Access Analyzer                       │
```

## 기능 비교

| 기능 | 용도 | 적용 범위 |
|------|------|----------|
| Access Analyzer | 보안 분석, 정책 검증 | 계정/조직 |
| 교차 계정 역할 | 계정 간 접근 | 계정 간 |
| SCP | 조직 가드레일 | 조직/OU/계정 |
| 권한 경계 | 위임된 권한 제한 | 사용자/역할 |

## 엔터프라이즈 아키텍처 예시

```
┌─────────────────────────────────────────────────────────────┐
│                    AWS Organizations                         │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  ┌─────────────────────────────────────────────────────┐   │
│  │                    관리 계정                         │   │
│  │  - Organizations 관리                               │   │
│  │  - IAM Identity Center                              │   │
│  │  - 통합 결제                                        │   │
│  └─────────────────────────────────────────────────────┘   │
│                          │                                  │
│         ┌────────────────┼────────────────┐                │
│         ▼                ▼                ▼                │
│  ┌──────────────┐ ┌──────────────┐ ┌──────────────┐       │
│  │  보안 OU     │ │ 공유서비스 OU │ │  워크로드 OU │       │
│  │              │ │              │ │              │       │
│  │ - 로그 계정  │ │ - 네트워크   │ │ - 프로덕션   │       │
│  │ - 감사 계정  │ │ - 공유 도구  │ │ - 개발       │       │
│  │ - 보안 도구  │ │              │ │ - 테스트     │       │
│  └──────────────┘ └──────────────┘ └──────────────┘       │
│        │                │                │                 │
│        ▼                ▼                ▼                 │
│    [엄격한 SCP]    [표준 SCP]      [환경별 SCP]           │
│                                                             │
└─────────────────────────────────────────────────────────────┘

추가 구성:
- IAM Access Analyzer: 모든 리전에서 활성화
- 교차 계정 역할: 중앙 집중식 로깅, 감사용
- CloudTrail: 조직 전체 트레일
```
