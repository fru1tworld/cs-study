## 빠른 보안 체크리스트

### 즉시 조치 필요

```
□ 루트 사용자 MFA 활성화
□ 루트 액세스 키 없음 확인
□ 모든 사용자 MFA 활성화
□ 미사용 자격 증명 비활성화
```

### 단기 (1주 내)

```
□ 비밀번호 정책 설정
□ IAM Access Analyzer 활성화
□ CloudTrail 활성화
□ 기본 알림 설정
```

### 중기 (1개월 내)

```
□ 연합 인증 검토
□ 권한 검토 및 최소화
□ 액세스 키 교체 계획
□ SCP 가드레일 설정
```

### 정기 (분기별)

```
□ 자격 증명 보고서 검토
□ 미사용 권한 제거
□ 액세스 키 교체
□ 정책 검증
```

## 보안 도구

| 도구 | 용도 |
|------|------|
| IAM Access Analyzer | 외부 접근, 정책 검증, 정책 생성 |
| Credential Report | 자격 증명 상태 감사 |
| CloudTrail | API 호출 기록 |
| AWS Config | 규정 준수 모니터링 |
| GuardDuty | 위협 탐지 |
| Security Hub | 보안 현황 통합 |

## 일반적인 보안 위협

### 1. 자격 증명 노출

```
위협: 액세스 키가 공개 저장소에 노출
대응:
- 즉시 키 비활성화/삭제
- CloudTrail 로그 검토
- 의심스러운 리소스 확인
```

### 2. 과도한 권한

```
위협: 필요 이상의 권한이 부여됨
대응:
- IAM Access Analyzer로 분석
- 마지막 접근 정보 확인
- 미사용 권한 제거
```

### 3. MFA 미사용

```
위협: 비밀번호만으로 접근 가능
대응:
- MFA 강제 정책 적용
- 정기적인 MFA 사용 여부 감사
```

### 4. 키 미교체

```
위협: 오래된 액세스 키 사용
대응:
- 90일 교체 정책
- AWS Config 규칙으로 모니터링
- 자동 알림 설정
```
