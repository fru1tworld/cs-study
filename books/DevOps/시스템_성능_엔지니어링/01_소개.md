# 1장 소개

## 1.1 시스템 성능

시스템 성능 분석은 소프트웨어와 하드웨어의 모든 핵심 구성 요소를 포괄하여 컴퓨터 시스템 전체의

성능을 연구화는 과정

데이터가 지나는 경로의 모든 요소, 저장 장치에서부터 애플리케이션 소프트웨어에 이르기까지, 이 모든

요소가 성능에 영향을 미칠 수 있고 성능 분석의 대상

대상 애플리케이션의 경로가 포함된 다이어그램이 없다면 직접 그려보는 것이 좋음

이것의 목적은 컴퓨팅 비용을 절감하여, 비효율성을 제거하고, 시스템 스루풋을 향상시키며 시스템

전반을 튜닝함으로써 비용을 절감할 수 있음

소프트웨어에서 말하는 풀스택과 시스템 성능에서 말하는 풀스택이 다름 하드웨어까지 포함함 심지어

컴파일러도 있는데 이조차 성능에 영향을 미칠 수 있기 때문임

수용량 계획은 앞서 언급한 여러 활동 중 일부를 포함하는 개념으로, 설계 단게에서는 개발 중인

소프트웨어의 자원 사용량을 분석하여, 설계가 목표 성능을 충족할 수 있는지 평가함

## 1.5 성능 분석의 어려움

### 1.5.1 주관성
