# Shardcake 소개

> 원본: https://devsisters.github.io/shardcake/about/

---

**Shardcake**는 인기 게임 [**쿠키런: 킹덤**](https://www.cookierun-kingdom.com)을 만든 [**Devsisters**](https://www.devsisters.com) 팀이 개발한 프로젝트입니다. 2022년 3월부터 프로덕션에서 문제없이 잘 쓰고 있습니다.

한국에 있다면 [채용](https://careers.devsisters.com/ko/position?occupations=%EA%B0%9C%EB%B0%9C) 중입니다!

**Shardcake**는 [**Akka Cluster Sharding**](https://doc.akka.io/docs/akka/current/typed/cluster-sharding.html)을 여러 해 사용한 경험을 바탕으로 만들었습니다. 코드는 Akka와 공통점이 전혀 없지만, 몇몇 아이디어와 개념은 Akka에서 영감을 받았습니다.
