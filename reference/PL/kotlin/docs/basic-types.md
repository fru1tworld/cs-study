# Kotlin 기본 타입 개요

Kotlin의 기본 타입은 다음과 같은 범주로 구성됩니다:

## 숫자 타입

### 정수 타입
- `Byte` - 8비트
- `Short` - 16비트
- `Int` - 32비트
- `Long` - 64비트

### 부호 없는 정수 타입
- `UByte` - 8비트
- `UShort` - 16비트
- `UInt` - 32비트
- `ULong` - 64비트

### 부동 소수점 타입
- `Float` - 32비트
- `Double` - 64비트

## 불리언 타입
- `Boolean` - `true` 또는 `false`

## 문자 타입
- `Char` - 단일 문자

## 문자열 타입
- `String` - 문자 시퀀스
