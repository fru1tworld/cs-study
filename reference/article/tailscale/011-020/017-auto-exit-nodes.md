# 자동 출구 노드(Auto Exit Nodes) 소개

게시일: 2024년 5월 30일

작성자: Kabir Sikand, Jairo Camacho, Claire Wang, Adrian Dewhurst

카테고리: 제품

---

자동 출구 노드(Auto Exit Nodes)는 사용자를 가장 가깝고 성능이 좋은 출구 노드에 연결하여 공용 인터넷에 접속할 수 있게 해주는 Tailscale의 새로운 기능입니다. 직원들은 사무실, 코워킹 스페이스, 단골 카페, 또는 출장 중 공항이나 호텔 등 다양한 장소에서 일합니다. 자동 출구 노드를 사용하면 고용주는 직원들이 최상의 연결을 사용할 수 있도록 보장하여 원격 근무를 지원할 수 있으며, 직원들이 보안이 취약한 우회 방법을 찾지 않아도 됩니다.

## 조직 전체로 확장 가능하도록 설계

올해 초, 우리는 리전 라우팅(Regional Routing) 기능을 출시했습니다. 이 기능은 조직이 사용 가능한 인프라 전반에 걸쳐 트래픽을 효율적으로 라우팅하고 부하를 분산하여 앱 커넥터(App Connectors)와 서브넷 라우터(Subnet Routers)를 전 세계적으로 확장할 수 있도록 도와줍니다. 이제 조직은 리전 라우팅의 강력한 기능을 출구 노드까지 확장하여 사용자의 지리적 위치에 관계없이 안정적이고 성능 중심의 경험을 제공할 수 있습니다.

[이미지 설명: 자동 출구 노드는 전 세계 인력에게 안전한 인터넷 접속을 확장하는 데 도움이 됩니다.]

## 아웃바운드 트래픽 보안을 위한 간소화된 워크플로우

MacOS, Windows, iOS, 또는 Android(곧 출시 예정) 클라이언트를 사용하는 고객은 Tailscale이 권장하는 출구 노드를 선택할 수 있습니다. Linux 또는 CLI를 사용하는 고객의 경우, `tailscale exit-node suggest`와 같은 `exit-node suggest` 명령을 입력하여 현재 권장되는 출구 노드를 선택할 수 있습니다. 권장 사항은 측정된 지연 시간(latency)을 기반으로 하며, 새로운 노드를 선택하거나 자동 출구 노드 옵션을 다시 선택할 때까지 해당 출구 노드에 연결된 상태를 유지합니다.

## Mullvad와 연동

자동 출구 노드는 Mullvad 출구 노드도 추천할 수 있습니다. Mullvad 애드온을 구독한 테일넷(tailnet)의 경우, Tailscale은 IP 지리적 위치와 가장 가까운 Mullvad 서버 위치를 조합하여 출구 노드를 추천하며, 코로케이션(colo-local) Mullvad 부하 분산 힌트를 고려합니다.

## 강제 출구 노드로 인력 보안 확보

고객들은 곧 강제 출구 노드(Forced Exit Nodes)를 사용하여 모든 플랫폼에서 사용자 입력 없이 전체 인력을 보안할 수 있게 됩니다. Enterprise 및 Premium 플랜의 테일넷은 JAMF, Microsoft Intune, Kandji, TinyMDM 등과 같은 모바일 기기 관리(MDM) 도구를 사용하여 Tailscale이 권장하는 출구 노드를 사용하거나 전체 플릿에서 특정 출구 노드를 사용하도록 구성할 수 있습니다. 노드를 사용할 수 없게 되면 Tailscale은 자동으로 다음 사용 가능한 노드에 재연결을 시도합니다.

## 하이브리드 인력을 위한 희소식

공용 인터넷에는 모든 기업에 대한 위협이 도사리고 있습니다. 하이브리드 인력은 특히 위험에 노출되어 있는데, 직원과 계약자들이 의심스러운 연결과 네트워크를 통해 SaaS 애플리케이션, 파트너 포털 등과 같은 회사 리소스에 지속적으로 접근하기 때문입니다. 자동 출구 노드를 사용하면 Tailscale 고객은 원격 직원들이 신뢰할 수 있는 송신 지점(egress points)을 사용하여 민감한 리소스에 접근하도록 활성화하거나 강제하여 신뢰할 수 없는 네트워크의 활성 위협에 대비할 수 있습니다. 출구 노드에 대한 연결은 사용자의 로컬 네트워크에 관계없이 직접 또는 릴레이된 WireGuard 터널을 통해 보안됩니다.

전 세계, 하이브리드, 또는 원격 인력이 있다면 여러 가용 영역(availability zones), 여러 클라우드, 지사 등에 출구 노드를 설정하세요. 그런 다음 MDM 솔루션을 사용하여 자동 출구 노드 사용을 활성화하면 직원들을 수동 입력 없이 가장 가까운 사용 가능한 출구 노드로 지능적으로 라우팅할 수 있습니다.

## 지금 바로 시작하세요
