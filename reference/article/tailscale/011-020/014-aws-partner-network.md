Tailscale은 이제 AWS 파트너 네트워크(AWS Partner Network)의 공인 멤버가 되었습니다. 이를 통해 사용자들은 AWS 환경에서 보안이 강화된 WireGuard® 기반 메시 네트워크를 더욱 쉽게 배포하고 구성할 수 있게 되었습니다. 몇 대의 클라우드 서버를 운영하는 소규모 팀부터 복잡한 엔터프라이즈 배포 환경을 갖춘 기업까지, Tailscale은 원격 접속 연결을 간소화하고, 보안 관행을 개선하며, 컴플라이언스 의무를 체계적으로 관리할 수 있도록 도와줍니다.

Tailscale과 AWS를 함께 사용하는 방법에 대해 더 알고 싶으신가요? 4월 30일에 AWS 시니어 솔루션 아키텍트 Max Paz와 공동 주최하는 웨비나 "Tailscale & AWS: 엔터프라이즈 클라우드 네트워킹 간소화(Tailscale & AWS: Enterprise Cloud Networking Simplified)"에 참여해 주세요. Tailscale이 AWS 환경과 연계하여 연결성과 보안을 어떻게 간소화할 수 있는지에 대한 실용적인 팁과 데모를 제공해 드릴 예정입니다.

## Tailscale + AWS

Tailscale은 이제 AWS 마켓플레이스(AWS Marketplace)에 등록되어 있습니다. 이미 AWS 생태계를 사용 중인 사용자들은 빠른 배포, 통합 청구, 그리고 AWS에 선불로 약정한 금액을 Tailscale 구매에 사용할 수 있는 혜택을 누릴 수 있습니다.

클라우드 워크로드에 원격 접속하기 위해 레거시 VPN을 사용해 본 경험이 있다면, 그 단점들을 이미 잘 알고 계실 것입니다. 잦은 연결 끊김부터 비효율적인 백홀(backhaul) 라우팅으로 인한 높은 지연 시간까지, 전통적인 VPN 아키텍처의 한계는 중요한 클라우드 호스팅 서비스에 연결하는 사용자들과 이를 지원하려는 IT 리더들 모두에게 지속적인 도전 과제를 안겨줍니다.

이와 대조적으로, Tailscale은 클라우드의 유연성을 염두에 두고 설계되었습니다. 그리고 수년간 고객들이 AWS 환경에서 배포하는 것을 도우면서, 우리는 클라이언트가 해당 환경에서 더욱 잘 작동하도록 지속적으로 개선하고 최적화해 왔습니다. 그 결과물은 AWS 워크플로우를 보완하는 완전한 기능을 갖춘 솔루션입니다. 이 솔루션은 안전한 원격 접속뿐만 아니라 가용 영역(zone) 간 고가용성, 서브넷 라우팅, 그리고 세분화된 세그먼테이션(granular segmentation) 및 마이크로 세그먼테이션(micro-segmentation)까지 제공합니다.

## 추가 리소스

AWS와 Tailscale 사용자들을 위해, 문서, 고객 사례 연구, 지식 베이스 가이드, 블로그, AWS 레퍼런스 아키텍처 등에 대한 링크를 모아놓은 통합 랜딩 페이지를 게시했습니다. 이러한 리소스들은 AWS 리소스에 대한 안전한 원격 접속을 위해 Tailscale을 빠르게 통합하고, 더 복잡한 요구 사항에 대한 솔루션을 개발하는 데 도움이 될 수 있습니다.

질문이 있으시면 aws-marketplace@tailscale.com으로 연락해 주시면, 저희 팀이 사용 사례를 함께 살펴보고 모범 사례를 제안해 드리겠습니다.

마지막으로, 4월 30일에 열리는 "Tailscale & AWS: 엔터프라이즈 클라우드 네트워킹 간소화" 웨비나에 꼭 참여해 주세요. Tailscale 솔루션 엔지니어 팀과 AWS 솔루션 아키텍트가 공동 주최합니다. 참석은 무료이며, 사전 등록이 필요합니다.
