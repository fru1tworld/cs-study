# 오픈소스 GitHub 조직을 위한 무료 Tailscale 플랜

게시일: 2021년 9월 17일

저자: Ross Zurowski, Sonia Appasamy, David Crawshaw, Brad Fitzpatrick

---

Tailscale은 오픈소스를 사랑합니다. 우리는 프로젝트를 공개적으로 개발하고, 전 세계의 개인 및 조직과 협업하는 것이 얼마나 어려운 일인지 잘 알고 있습니다.

오픈소스 프로젝트에 Tailscale을 사용하는 GitHub 조직에게 Tailscale이 무료라는 것을 발표하게 되어 기쁩니다. 그리고 Tailscale은 연결을 만드는 데 뛰어나기 때문에, GitHub 조직 계정을 사용하여 협업하는 친구와 가족도 이 무료 플랜의 혜택을 받을 수 있습니다.

## 새로운 Community on GitHub 플랜

OSI 라이선스를 사용하는 오픈소스 프로젝트가 GitHub 조직을 사용하고 있다면, 해당 조직에서 Tailscale을 무료로 사용할 수 있습니다. (현재로서는 사용자 관리 방식으로 인해 GitHub 조직에만 이 서비스를 제공하고 있습니다. 다른 ID 제공자를 사용하시는 경우 알려주세요.)

새로운 Community on GitHub 플랜에서는 다음을 제공합니다:

- 최대 25명의 사용자
- 사용자당 5대의 기기
- 1개의 서브넷 라우터 (subnet router)
- 2명의 관리자 사용자
- ACL 정책에서 2명의 고유 사용자
- 커뮤니티 지원

Tailscale이 계속 발전함에 따라 시간이 지나면서 더 많은 기능을 추가할 수 있기를 바랍니다. 더 많은 기기가 필요하거나 이러한 제한으로 인해 어려움을 겪고 계신가요? 알려주세요.

이 플랜은 가족과 친구에게도 제공됩니다. 아버지의 사진 서버에 접속하고, 며느리의 새 앱에 피드백을 제공하고, 이웃의 공용 진입로 웹캠을 확인해보세요.

Tailscale은 또한 자선 단체, 비영리 단체, 교육 기관에 정가의 50% 할인을 제공합니다. 이 할인을 받으시려면 저희에게 연락해 주세요.

## Personal 플랜은 항상 무료입니다

지난달 새로운 가격 정책을 도입한 후, 여러분의 목소리를 들었습니다. 무료 Personal 티어를 변경할 것을 우려하셨습니다. 그것은 전혀 계획에 없습니다.

개인 기기에서 Tailscale을 개인적으로 사용하기 위한 Personal 플랜은 항상 무료입니다. Pi-hole이나 Minecraft 서버, 또는 강아지 캠에 접속하는 것이 얼마나 어려운 일인지 기억하고 있으며, 그런 어려움을 누구에게도 주고 싶지 않습니다. Personal 플랜은 사라지지 않습니다.

## Community on GitHub 시작하기

GitHub 조직을 새로운 무료 Community on GitHub 플랜으로 업그레이드하려면, 관리자 콘솔의 Billing 설정에서 새 플랜을 선택하세요. 우리는 오픈소스를 지원하는 것을 자랑스럽게 생각합니다.
