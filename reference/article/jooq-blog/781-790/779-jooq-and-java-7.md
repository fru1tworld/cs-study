# jOOQ와 Java 7

> 원문: https://blog.jooq.org/jooq-and-java-7/

2011년 8월 4일 lukaseder 작성

인터넷에서 Java 7의 버그로 인해 Lucene과 Solr, 그리고 다른 소프트웨어들이 크래시되는 것에 대한 블로그 글들이 점점 더 많이 올라오고 있습니다. 방금 Lucene 개발자가 쓴 이 흥미로운 글을 발견했습니다:

http://blog.thetaphi.de/2011/07/real-story-behind-java-7-ga-bugs.html

Java 7을 사용하는 것이 jOOQ 사용자들에게 어떤 영향을 미칠지 궁금합니다. 지금까지 보고된 문제는 없지만 :-) 그것이 아무 의미가 없을 수도 있습니다. 반면에, 제정신인 사람이라면 전체 JDK 플랫폼의 첫 번째 새로운 메이저 릴리스로 운영 시스템을 즉시 전환하지는 않을 것입니다.
