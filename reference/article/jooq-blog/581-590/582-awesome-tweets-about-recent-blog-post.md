# 최근 블로그 포스트에 대한 놀라운 트윗들

> 원문: https://blog.jooq.org/awesome-tweets-about-recent-blog-post/

블로그 글을 작성할 때, 대부분의 사람들은 훌륭한 콘텐츠를 만들기 위해 노력합니다. 훌륭한 콘텐츠를 만드는 것은 매우 어렵습니다. 대부분의 콘텐츠는 다음과 같은 이유로 그저 그렇습니다:

- 너무 틈새(niche) 주제여서
- 관련성이 없어서
- 그냥 간과되거나
- 지루하거나
- 홍보성이어서

하지만 때때로 훌륭한 콘텐츠가 나타나기도 합니다. 제 이전 블로그 글 ["Java 개발자들이 SQL 작성 시 저지르는 10가지 흔한 실수"](https://blog.jooq.org/2013/07/30/10-common-mistakes-java-developers-make-when-writing-sql/)가 바로 그랬습니다.

훌륭한 콘텐츠를 어떻게 알아볼 수 있을까요? 트위터를 확인하면 됩니다.
