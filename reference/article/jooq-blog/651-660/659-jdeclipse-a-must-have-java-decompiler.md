# JDEclipse: 꼭 필요한 Java 디컴파일러

> 원문: https://blog.jooq.org/jdeclipse-a-must-have-java-decompiler/

최근 저는 매우 유용한 것을 발견했습니다. 바로 소스 코드를 쉽게 찾거나 Eclipse에 연결할 수 없는 서드파티 라이브러리를 다룰 때 사용할 수 있는 Eclipse 플러그인입니다.

JDEclipse는 바이트코드를 Java 코드로 디컴파일하면서 줄 번호를 올바른 위치에 유지해 줍니다. 따라서 디컴파일된 소스 코드를 통해 디버깅할 수 있고, 기능하는 브레이크포인트를 설정할 수 있습니다!

자세한 내용은 여기를 참조하세요: http://mchr3k.github.com/jdeclipse-realign/

물론 라이선스가 허용하지 않는 코드를 디컴파일해서는 안 됩니다...
