# iciql, H2의 JaQu 포크

> 원문: https://blog.jooq.org/iciql-a-fork-of-h2s-jaqu/

2011년 8월 6일, lukaseder

SQL 시뮬레이션 Fluent API 분야에서 새로운 경쟁자를 발견하는 것처럼, 저는 iciql이라는 새로운 프로젝트를 발견했습니다. http://iciql.com/ 에서 확인할 수 있습니다.

iciql은 H2의 JaQu 라이브러리에 이전에 기여했던 James Moger가 만들었습니다. JaQu는 H2 데이터베이스의 부산물로, 현재 H2 로드맵에서 낮은 우선순위로 인해 어려움을 겪고 있습니다. Moger가 프로젝트를 포크하기로 한 결정은 JaQu가 H2의 개발 계획 내에서 받고 있던 제한된 개발 관심에 의해 동기부여된 것입니다. 이는 좋은 움직임입니다 - 라이브러리에 대한 기여 모멘텀을 독립적으로 계속 개발하면서 유지하는 것이니까요.

그의 행운을 빕니다!
