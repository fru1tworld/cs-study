# jOOQ 뉴스레터: 2014년 2월 12일

> 원문: https://blog.jooq.org/jooq-newsletter-february-12-2014/

## 오늘의 트윗

두 가지 주목할 만한 트윗이 있었습니다:

1. Breandan Considine는 jOOQ 덕분에 "10파운드 이상 감량"하고 "생각하는 속도로 쿼리를 작성"할 수 있게 되었다고 트윗했습니다. 코딩하느라 밥 먹는 것을 잊었다는 유머러스한 언급도 함께였습니다.

2. Chris Martin은 jOOQ 3.3의 새로운 SEEK 메서드에 대한 열정을 표현했습니다.

## RedHat opensource.com에 게재된 Data Geekery 비즈니스 사례

저희 팀은 RedHat 플랫폼에 오픈 소스 소프트웨어로 수익 기반 모델로 전환하는 기업을 위한 5가지 교훈에 대한 글을 게시했습니다. 이 글에서는 저희 회사의 ASL 2.0과 상용 듀얼 라이선싱 접근 방식을 언급했습니다.

잠재적인 통합 프로젝트로 언급된 것들:
- Apache Gora™
- Apache CloudStack™

## 커뮤니티 활동

주목할 만한 커뮤니티 기여:

- Loiane Groner (IBM 컨설턴트이자 저자)가 jOOQ 영속성에 대한 포르투갈어 튜토리얼을 작성했습니다
- Ben Hood가 jOOQ에서 영감을 받아 Apache Cassandra를 위한 DSL인 CQLC를 만들었습니다
- Breandan Considine가 Gradle과 IntelliJ IDEA를 사용한 jOOQ에 대한 비디오 튜토리얼을 제작했습니다

## Java 존 – Java 8

"Java 8 Friday Goodies"라는 새로운 블로그 시리즈가 발표되었습니다. 이 시리즈는 Java 8의 일상적인 개선 사항과 변경 사항을 다루며, 다음 주제들을 포함합니다:
- java.io가 드디어 멋져졌습니다
- 람다와 XML
- 새로운 New I/O API들
- 람다와 정렬
- 람다와 SQL

독자들에게 Java 8 카테고리를 구독할 것을 권장했습니다.
