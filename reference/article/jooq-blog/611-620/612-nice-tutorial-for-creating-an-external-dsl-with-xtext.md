# Xtext를 사용한 외부 DSL 생성에 관한 훌륭한 튜토리얼

> 원문: https://blog.jooq.org/nice-tutorial-for-creating-an-external-dsl-with-xtext/

게시일: 2013년 7월 21일, 작성자: lukaseder

저는 보통 내부 DSL(Domain-Specific Language, 도메인 특화 언어)에 대해 블로그에 글을 씁니다. jOOQ가 대표적인 예시입니다. 하지만 외부 DSL도 마찬가지로 매력적입니다.

외부 DSL은 호스트 언어(예: Java)에 의존하지 않기 때문에, 문법과 표현력 면에서 훨씬 더 풍부할 수 있습니다. 그러나 외부 DSL은 파싱, 컴파일, 실행을 위한 전용 도구가 필요합니다.

Eclipse의 DSL 개발 프레임워크인 Xtext를 사용하여 외부 DSL을 생성하는 튜토리얼이 있습니다. 이 특정 튜토리얼은 비디오 게임 제어 흐름을 위한 커스텀 언어를 구축하는 방법을 보여주며, Xtext가 구문 강조(syntax highlighting)와 오류 검사를 포함한 IDE 기능을 자동으로 제공하는 방법을 소개합니다.
