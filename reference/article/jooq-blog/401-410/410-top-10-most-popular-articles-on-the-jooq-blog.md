# jOOQ 블로그에서 가장 인기 있는 기사 Top 10

> 원문: https://blog.jooq.org/top-10-most-popular-articles-on-the-jooq-blog/

*2014년 9월 22일, Lukas Eder 작성*

이 글은 당시 jOOQ 블로그에서 가장 많이 읽힌 10개의 기사를 소개합니다.

## 목록

1. "Top 10 Very Very VERY Important Topics to Discuss" — Reddit의 r/programming을 풍자한 글로, 기술 토론에서 바이크셰딩(사소한 것에 대한 과도한 논쟁)이 어떻게 지배적인지를 강조합니다.

2. "10 Subtle Best Practices when Coding Java" — Java 개발에 적용할 수 있는 덜 알려진 프로그래밍 조언을 탐구합니다.

3. "10 Common Mistakes Java Developers Make when Writing SQL" — Java 애플리케이션에서 데이터베이스 쿼리를 작성하는 개발자에게 필수적인 읽을거리입니다.

4. "SQL Trick: row_number() is to SELECT what dense_rank() is to SELECT DISTINCT" — 윈도우 함수와 그 실용적인 활용법을 설명하는 높은 순위의 글입니다.

5. "Why You Should NOT Implement Layered Architectures" — 아키텍처 정통주의에 의문을 제기하고 개발자들이 표준 패턴을 재고하도록 장려하는 도발적인 기사입니다.

6. "MIT Prof. Michael Stonebraker: 'The Traditional RDBMS Wisdom is All Wrong'" — 관계형 데이터베이스 관리 시스템의 미래에 대한 Stonebraker의 논쟁적인 주장을 다룹니다.

7. "The 10 Most Popular DB Engines (SQL and NoSQL)" — 데이터베이스 인기 순위를 분석하고 기술 채택의 변화를 관찰합니다.

8. "Does Java 8 Still Need LINQ? Or is it Better than LINQ?" — Java 8의 Streams API와 Microsoft의 LINQ 프레임워크를 비교합니다.

9. "10 More Common Mistakes Java Developers Make when Writing SQL" — 추가적인 SQL 함정을 다루는 속편입니다.

10. "The Java Fluent API Designer Crash Course" — Java에서 도메인 특화 언어를 만들기 위한 설계 원칙을 설명하며, jOOQ의 아키텍처에 직접적으로 적용됩니다.
