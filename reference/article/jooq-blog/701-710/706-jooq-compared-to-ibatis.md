# jOOQ와 iBatis 비교

> 원문: https://blog.jooq.org/jooq-compared-to-ibatis/

jOOQ를 iBATIS와 비교하는 jOOQ 사용자의 좋은 피드백입니다:

> "jOOQ 최고입니다! 저희는 jOOQ로 대규모 ETL 서비스를 구현했는데, 더할 나위 없이 만족하고 있습니다. 저는 iBATIS를 사용해 본 적이 있는데, jOOQ는 방대한 XML과 타입 안전성 부재 없이도 저수준 SQL의 모든 장점을 동일하게 제공합니다. [...] jOOQ의 가장 좋은 기능 중 하나는 바로 이것입니다: 복잡하거나 동적으로 생성되는 쿼리를 제어된 방식으로 조립하고 Java 컴파일러로 검증할 수 있어서, 데이터베이스가 던지는 모호한 구문 오류로부터 벗어날 수 있고, 스키마가 변경될 때 컴파일 타임 회귀 커버리지를 제공합니다."

Stack Overflow 사용자 dacc의 [Stack Overflow에서의 피드백](https://stackoverflow.com/a/4522144/521799)
