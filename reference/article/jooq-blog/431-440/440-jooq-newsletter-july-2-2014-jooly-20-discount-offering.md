## 커뮤니티 존

### JINQ와 jOOQ 통합

JINQ가 jOOQ와의 통합을 발표했습니다. 이를 통해 Java 8 Streams API를 사용한 쿼리를 SQL로 백엔드 처리할 수 있게 되었습니다.

### jOOQ와 Flyway 워크샵

Instil Software가 2014년 7월 16일 벨파스트에서 jOOQ와 Flyway 워크샵을 개최했으며, 이 워크샵은 매진되었습니다.

### ZeroTurnaround Java 도구 설문조사

jOOQ가 ZeroTurnaround의 2014년 Java 도구 및 기술 현황(Java Tools and Technologies Landscape) 설문조사에 포함되었습니다. 이 설문조사는 2,164명의 Java 전문가를 대상으로 진행되었으며, jOOQ는 MyBatis와 같은 기존의 확립된 경쟁 제품들과 함께 순위에 올랐습니다.
