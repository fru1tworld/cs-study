# 필수 Eclipse 플러그인: AnyEdit Tools

> 원문: https://blog.jooq.org/must-have-eclipse-plugin-anyedit-tools/

새로운 Eclipse를 설치할 때마다, 저는 보통 이 사랑스러운 작은 플러그인이 기본으로 포함되어 있지 않다는 것을 깨닫기 전까지 뭔가 허전한 느낌을 받습니다.

이 플러그인은 다음과 같은 편리한 변환 및 비교 기능을 제공합니다:

- 탭과 공백 변환
- 문자와 HTML 엔티티 변환
- 카멜 케이스와 언더스코어 변환
- 변환: 대문자로 시작
- 변환: 대소문자 반전
- 변환: 대문자로 변환
- 변환: 소문자로 변환
- 변환: Base64 인코딩/디코딩
- 클립보드와 비교/교체
- 워크스페이스 파일과 비교/교체
- 외부 파일과 비교/교체
- 열린 에디터와 비교/교체

이러한 기능 대부분은 단축키를 통해서도 사용할 수 있습니다.
