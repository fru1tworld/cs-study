# jOOQ와 Hibernate, 토론

> 원문: https://blog.jooq.org/jooq-and-hibernate-a-discussion/

이 짧은 블로그 글은 jOOQ와 Hibernate를 비교하는 Reddit 토론을 참조합니다. 원래의 비교는 "다소 감정적이고 아마도 정말 객관적이지 않았다"고 저자는 설명하지만, 데이터베이스 추상화 계층과 SQL에 관심 있는 사람들에게 탐구할 가치가 있는 흥미로운 대화를 촉발했다고 언급합니다.

토론에서 주목할 만한 댓글을 강조합니다:

> "jOOQ보다 SQL에 더 잘 매핑할 수 있는 데이터베이스 추상화 계층을 찾기는 어려울 것입니다. jOOQ는 사실상 SQL 그 자체입니다."

이 글은 jOOQ가 데이터베이스 작업에 SQL 중심적 접근 방식을 취하며, Hibernate와 같은 전통적인 ORM 도구와는 확실히 다른 위치에 있음을 강조합니다. 추상화 패턴 뒤에 SQL을 숨기는 대신, jOOQ는 Java 기반 쿼리 빌딩 기능을 제공하면서 SQL을 주요 인터페이스로 유지합니다.
