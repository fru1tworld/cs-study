# Vaadin, 매우 좋은 웹 프레임워크

> 원문: https://blog.jooq.org/vaadin-a-very-nice-web-framework/

최근 [Play! 프레임워크](https://www.playframework.com/)에 대한 열풍은 많은 사람들로 하여금 "바로 이거야! 이렇게 해야 하는 거야"라고 생각하게 만든 것 같습니다. 그리고 Play! 튜토리얼을 직접 따라해 본 후, 저도 즉시 동의했습니다. 웹 개발은 Play!에서처럼 쉬워야 합니다.

하지만 Play!는 요청-응답 방식의 HTML 기반 웹사이트에서의 일상적인 작업을 단순화하는 것에 불과합니다. 세상에는 상태 기반(stateful)이고 컴포넌트 기반의 사고방식으로 작업을 단순화하는 것을 목표로 하는 다른 프레임워크들이 있으며, 이는 웹 애플리케이션에서 자주 해당되는 경우입니다. 그래서 저는 정확히 그런 일을 하는 이 보석 같은 프레임워크를 발견하게 되었습니다:

https://vaadin.com/

Vaadin은 GWT(Google Web Toolkit) 위에 구축되어 있으며, 이는 웹 프레임워크를 위한 견고한 기반입니다. 무료 오픈소스 버전과 더 나은 지원을 받을 수 있는 "프로 버전"이 있습니다. 두 경우 모두, 사용자들은 훌륭한 애드온 세트를 활용할 수 있으며, 커뮤니티도 무시할 수 없는 규모인 것 같습니다:

https://stackoverflow.com/questions/tagged/vaadin

jOOQ-웹 통합을 위한 또 다른 후보가 될 수 있을까요?
