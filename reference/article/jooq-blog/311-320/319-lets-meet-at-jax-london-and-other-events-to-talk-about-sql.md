# JAX London과 기타 이벤트에서 SQL에 대해 이야기하자

> 원문: https://blog.jooq.org/lets-meet-at-jax-london-and-other-events-to-talk-about-sql/

Java/SQL 개발자들에게 흥미로운 시간이 다가오고 있습니다. SQL은 여전히 강력하고 인기가 높습니다. SQL은 관계형 데이터베이스뿐만 아니라 빅데이터 스토리지 시스템에도 적합한 언어입니다. 바로 이번 주에 Pivotal은 Hadoop Native SQL 전략의 일환으로 HAWQ와 MADlib을 오픈 소스화한다고 발표했습니다.

Data Geekery에서 우리는 jOOQ를 통해 SQL을 홍보하고 있습니다 - jOOQ는 Java를 위한 타입 세이프한 임베디드 SQL입니다. 이제 새로운 고급 SQL 워크숍도 제공하고 있으며, 이 워크숍은 공개 행사와 사내 교육 형태로 모두 이용 가능합니다. 여기에서 확인해 보세요.

앞으로 몇 달 동안 우리는 다시 유럽 투어를 진행할 예정이며, 여러 장소 중에서 특히 JAX London에서 공식 미디어 파트너로서 발표를 할 것입니다.

저희 발표 외에도 다음의 흥미로운 발표들을 놓치지 마세요:

- "2000 Lines of Java or 50 Lines of SQL? The Choice is Yours" (저희 발표)
- "Coding for Desktop and Mobile with HTML5 and Java EE 7" (Geertjan Wielenga)
- "Design & Implementation of Microservices" (James Lewis)
- "Designing and Operating User-Centered Digital Services" (Jeff Sussna)
- "Distributed Systems in one Lesson" (Tim Berglund)
