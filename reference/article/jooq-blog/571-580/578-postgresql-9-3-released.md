# PostgreSQL 9.3 출시!

> 원문: https://blog.jooq.org/postgresql-9-3-released/

훌륭한 데이터베이스가 더욱 좋아졌습니다. PostgreSQL 9.3이 출시되었습니다. 신뢰성 개선 사항도 물론 중요하지만, jOOQ 사용자들에게 가장 흥미로운 부분은 새로운 SQL 구문입니다:

- 향상된 JSON 지원
- LATERAL JOIN (SQL:1999에서는 lateral 파생 테이블이라고 함)
- 구체화된 뷰(Materialised views)
- 업데이트 가능한 뷰(Updatable views)

전체 기능 목록은 PostgreSQL 위키의 "PostgreSQL 9.3의 새로운 기능"에서 확인하세요.
