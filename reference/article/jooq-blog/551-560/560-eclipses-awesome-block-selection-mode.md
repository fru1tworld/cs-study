# Eclipse의 놀라운 블록 선택 모드

> 원문: https://blog.jooq.org/eclipses-awesome-block-selection-mode/

이 글은 메뉴에 완전히 숨겨져 있어 과소평가되고 있지만, 수많은 상황에서 매우 유용한 놀라운 Eclipse 기능에 대한 것입니다. 바로 Windows에서 Alt-Shift-A로 토글할 수 있는 놀라운 "블록 선택 모드"입니다. 다음은 블록 선택 모드를 위한 예제 과제입니다:

Eclipse를 사용하여 Java 문자열 리터럴에 깔끔하게 포맷된 SQL 쿼리를 작성(복사-붙여넣기)할 수 있는 방법이 있을까요?

다음은 4단계 시각적 가이드로 답변을 제공합니다:

1단계: 붙여넣기

포맷된 SQL 문을 Java 파일에 그대로 붙여넣습니다.

2단계: 여는 따옴표 작성

강조 표시된 버튼(왼쪽에서 여섯 번째)을 주목하세요. 이것이 바로 놀라운 "블록 선택 모드"(Windows에서 Alt-Shift-A)입니다. 이 모드를 사용하면 선택한 각 줄의 동일한 위치에 여는 따옴표를 작성할 수 있습니다.

3단계: 닫는 따옴표와 연결 연산자 작성

닫는 따옴표와 연결 기호(+)에 대해서도 동일한 기법을 적용합니다.

4단계: 세미콜론 수정

여기에는 별도의 설명이 필요 없습니다.

결론: "놀라운 Eclipse 기능이죠, 그렇지 않나요?"
