# jOOQ 화요일: Raoul-Gabriel Urma가 Java 8과 영어의 공통점을 탐구하다

> 원문: https://blog.jooq.org/jooq-tuesdays-raoul-gabriel-urma-explores-what-java-8-and-english-have-in-common/

jOOQ 화요일 시리즈의 두 번째 에피소드에 오신 것을 환영합니다. 이 시리즈에서는 격월로 셋째 주 화요일마다 jOOQ 커뮤니티에서 흥미로운 작업을 하고 있다고 생각되는 분을 인터뷰합니다. 여기에는 jOOQ를 사용하거나 jOOQ의 생태계에 기여하는 사람들, 그리고 비슷한 제품을 개발하는 사람들이 포함됩니다.

저는 Raoul-Gabriel Urma와 이야기를 나눌 기회를 갖게 되어 매우 기쁩니다. Raoul은 Cambridge 대학에서 컴퓨터 과학 박사 학위를 받았습니다. 그의 연구는 프로그래밍 언어 진화에 중점을 두고 있습니다. 그는 또한 Java 8을 활용하는 개발자들의 생산성을 높이기 위한 접근법을 다루는 Manning 출판사의 최신 베스트셀러 "Java 8 in Action"의 공동 저자입니다. 또한 업계에서 다양한 역할을 수행해 왔습니다: Google, Oracle, eBay, Goldman Sachs에서 소프트웨어 엔지니어링 인턴으로 근무했고, 여러 스타트업을 공동 창업했으며, 기술 스타트업에 투자하기도 했습니다. 그는 Java, Scala, Python, JavaScript, Ruby를 포함한 다양한 언어로 일해왔습니다. 또한 20개 이상의 국제 컨퍼런스에서 연설했습니다.

## 안녕하세요 Raoul, 프로그래밍 언어 진화에 대한 박사 논문에 대해 말씀해 주시겠어요?

영어나 스페인어와 같은 자연어는 수세기에 걸쳐 진화합니다. 그 결과, 사회는 이러한 변화에 적응할 시간이 충분히 있습니다. 반면에 프로그래밍 언어는 거의 매년 진화합니다. 이것은 개발자들과 그들의 코드베이스에 지속적인 압박을 가합니다. 제 논문에서는 정적 타입 언어와 동적 타입 언어 모두에서 하위 호환성 문제의 원인을 찾는 데 중점을 둔 기계 지원 기술을 개발합니다. 이 주제에 흥미가 있으시다면, 저는 ECOOP 2015에서 이 분야에 관한 워크숍을 공동 주관하고 있습니다.

## 당신은 "Java 8 in Action"을 저술하셨습니다. 독자들은 이 책에서 무엇을 배울 수 있나요?

저는 Mario Fusco와 Alan Mycroft와 함께 이 책을 공동 저술했습니다. Java 8의 새로운 기능을 활용하는 방법을 가르쳐 드립니다. 하지만 단순히 언어 기능만을 담은 책이 아니라 오래도록 관련성을 유지할 수 있는 책을 쓰고 싶었습니다. 그래서 람다 표현식과 스트림 API 외에도 테스트와 디버깅, 리팩토링, 향상된 동시성, 함수형 프로그래밍 개념, 그리고 약간의 Scala까지 다룹니다.

## 책을 쓰면서 가장 크게 배운 점은 무엇인가요?

생각했던 것보다 항상 더 많은 시간이 걸린다는 것입니다.

## Java 8 도입에 있어 어떤 도전이 예상되시나요?

Java 8이 "Java를 다시 섹시하게" 만들었지만, 레거시 프레임워크와 코드베이스가 업계에서의 채택을 방해할 수 있습니다. 하지만 여러 설문 조사에 따르면 Java 8 채택률이 매우 높습니다. 컨퍼런스에서의 연설과 Java 8 교육 경험을 통해, Java 커뮤니티가 람다와 스트림 API 사용에 열광하고 있음을 알 수 있습니다. 저는 향후 5년 동안 함수형 스타일 프로그래밍이 확산될 것이라고 확신합니다.

## 도메인 특화 언어와 SQL에 대한 당신의 견해는 무엇인가요?

저는 jOOQ가 환상적인 프로젝트라고 생각합니다! 저는 비즈니스 도메인에 맞는 코드를 작성하는 것이 가독성과 유지보수에 도움이 되며, 궁극적으로 생산성에 유용하다고 생각합니다.

## 다양한 역할을 수행하시는 원동력은 무엇인가요?

저는 사람들의 삶과 지식에 기여하는 것을 좋아합니다!
