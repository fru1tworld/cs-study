# Play Framework와 함께 jOOQ 사용하기

> 원문: https://blog.jooq.org/using-jooq-with-play-framework/

jOOQ와 Play Framework를 함께 사용하는 것은 많은 Java 개발자들에게 단순함과 효율성 측면에서 자연스러운 조합처럼 보일 수 있습니다. 하지만 저는 개인적으로 Play에 대한 경험이 없습니다.

그래서 몇몇 매우 활발한 jOOQ 사용자들이 두 프레임워크를 통합하는 다양한 기법에 대해 논의한 GitHub 이슈를 참조해 주세요:

https://github.com/jOOQ/jOOQ/issues/768

그 토론에서 서드파티 플러그인이 탄생했으며, 여기에서 찾을 수 있습니다:

https://github.com/jaiew/play-jooq
