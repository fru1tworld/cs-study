# Eclipse 키보드 단축키를 쉽게 배우는 방법

> 원문: https://blog.jooq.org/learn-eclipse-keyboard-shortcuts-easily/

저는 저 자신의 생산성을 향상시키고 싶을 때 블로그를 검색하는 경우가 거의 없습니다. 적어도 제 일상적인 프로그래밍 작업에서는 말이죠. 하지만 이번 Eclipse 플러그인은 정말로 관심을 끄네요:

Mousefeed:
http://marketplace.eclipse.org/content/mousefeed

이 플러그인은 코드 인라인하기와 같은 리팩토링 작업에 중첩된 컨텍스트 메뉴를 사용하는 대신 "Alt-Shift-I" 단축키를 사용하도록 가르쳐줍니다. 이 플러그인이 하는 일은 여러분이 마우스로 메뉴를 사용할 때마다 거슬리는 팝업을 표시하는 것입니다. 해당 팝업은 동일한 작업에 대한 키보드 단축키가 있을 때 그 단축키를 보여줍니다. 이 팝업은 정말 짜증나서, 이 팝업이 뜨지 않게 하기 위해 키보드 단축키를 배우게 됩니다. 이러한 접근 방식의 좋은 점은 여러분이 자주 사용하는 기능에 대해서만 단축키를 배우게 된다는 것입니다. 제가 자주 사용하는 Ctrl-1이나 Ctrl-Space처럼요.

주의 사항: 일부 사용자들은 이 플러그인으로 인해 Eclipse Juno가 망가졌다고 보고했습니다. 사용에 따른 위험은 본인이 감수하세요!
