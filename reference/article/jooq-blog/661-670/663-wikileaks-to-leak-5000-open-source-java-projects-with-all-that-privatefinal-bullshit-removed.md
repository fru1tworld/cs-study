# 위키리크스, 그 짜증나는 Private/Final 키워드를 모두 제거한 5000개의 오픈소스 Java 프로젝트를 유출 예정

> 원문: https://blog.jooq.org/wikileaks-to-leak-5000-open-source-java-projects-with-all-that-privatefinal-bullshit-removed/

정말 웃기다. 심지어 4월 1일(만우절)에 쓴 글도 아니다!

[Steve Yegge의 원문에서 발췌](https://sites.google.com/site/steveyegge2/wikileaks):

> "전 세계의 Java 프로그래머들이 오늘 GMT 오전 8시 15분에 발표된 위키리크스 보도자료에 공황 상태에 빠졌다. 위키리크스는 수천 개의 오픈소스 Java 프로젝트 소스 코드를 재배포할 것이라고 발표했는데, 모든 접근 제어자를 'public'으로 변경하고 모든 클래스와 멤버에서 'final' 키워드를 제거할 것이라고 한다."

*(이 글은 Java의 캡슐화 관행과 오픈소스 프로젝트에서 `private`와 `final` 키워드의 과도한 사용에 대한 풍자적인 비평입니다.)*
