# jOOQ - Java와 SQL로 작업하는 대안적 방법

> 원문: https://blog.jooq.org/jooq-ein-alternativer-weg-mit-java-und-sql-zu-arbeiten/

iJUG e.V.에서 발행하는 독일어 잡지 [www.java-aktuell.de](http://www.java-aktuell.de)에 기사가 게재되었습니다.

기사에서 발췌:

> "Java에는 SQL의 표현력과 강력함을 직접 지원하는 표준 API가 없습니다. 모든 관심은 객체-관계형 매핑과 OQL, HQL, JPQL, CriteriaQuery와 같은 더 높은 추상화 수준에 집중되어 있습니다. jOOQ는 이 격차를 메우는 이중 라이선스 오픈소스 제품입니다. jOOQ는 SQL을 타입 안전한 도메인 특화 언어로 Java에 직접 구현하며, SQL과 벤더별 데이터베이스 기능이 중요한 Java 애플리케이션에 좋은 선택입니다. 이것은 현대적인 도메인 특화 언어가 SQL을 Java에 직접 임베드함으로써 개발자 생산성을 크게 높일 수 있음을 보여줍니다."

전체 기사는 이 블로그에서 무료 PDF로 다운로드할 수 있으며, SlideShare 프레젠테이션 버전도 제공됩니다.
