# Java가 JSR-341의 EL 3.0에 LINQ를 추가할 것인가?

> 원문: https://blog.jooq.org/will-java-add-linq-to-el-3-0-in-jsr-341/

Java가 Expression Language 3.0에 완전한 .NET 스타일의 LINQ 지원을 추가할 예정입니다.

적어도 JSR-341 웹사이트에 따르면 그렇습니다:

http://java.net/projects/el-spec/pages/CollectionOperations

하지만 몇 가지 의문점이 있습니다:

1. Microsoft가 전문가 그룹에 참여하고 있는가?
2. Microsoft가 LINQ 관련 특허를 Oracle에 양도하거나 라이선스를 부여할 것인가?
3. Oracle이 해당 특허에 이의를 제기할 것인가?
4. 왜 LINQ가 핵심 Java 플랫폼 자체에 통합되기 전에 Expression Language에 먼저 추가되는가?

메일링 리스트는 그다지 활발하지 않은 것 같고, 드래프트 스펙은 아직 상당히 초기 단계에 있습니다.

지켜봐야 할 것 같습니다. 이전에 이 주제에 대해 다룬 글이 있습니다: "언제 Java에 LINQ가 도입될까?"
