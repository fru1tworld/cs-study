# MySQL 8.0.34

날짜: 2023-12-18

Jepsen은 Kleppmann의 MySQL 격리 수준 연구를 재검토하여 버전 8.0.34에서 예상 밖의 동작을 발견했습니다.

## 발견된 문제

MySQL의 `REPEATABLE READ` 격리 수준이 여러 문제를 드러냈습니다:
- G2-item, G-single, lost update 현상 발생
- 내부 일관성 위반
- Monotonic Atomic View를 만족하지 않음

이 격리 수준은 Adya의 Repeatable Read 정의도, 모호한 ANSI SQL 정의도 충족하지 않습니다.

## AWS RDS MySQL

추가로 AWS RDS MySQL 클러스터가 `SERIALIZABLE` 격리 수준에서도 일관되게 직렬성을 위반하는 현상이 발견되었습니다.
