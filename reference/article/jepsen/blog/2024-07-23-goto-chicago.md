# GOTO Chicago 2024

날짜: 2024-07-22

Kyle Kingsbury는 2024년 10월 21일과 22일 GOTO Chicago에서 Jepsen의 성능 기술에 대해 발표할 예정입니다.

발표에서 다룰 주요 내용은 "높은 수준의 최적화와 낮은 수준의 성능 최적화를 포함하여 대규모 히스토리 검사를 다루기 쉽게 만드는" 기법들입니다.

구체적으로는 다음을 다룹니다:
- 병렬 처리
- 순수 함수
- 불변 자료구조
- 제거(deforestation)
- 비트셋
- 스레드 간 공유 회피
- 변경 가능한 배열로의 구조 압축
- 원시 박스의 동적 컴파일
- 매크로 반복 최적화
