# YugaByte DB 1.1.9

날짜: 2019-03-25

YugaByte와의 협력을 통해, Jepsen 분석은 YugaByte DB 버전 1.1.9에서 세 가지 중요한 안전성 취약점을 확인했습니다:

1. 정상 작동 조건에서 발생하는 읽기 왜곡(read skew)과 데이터 손상
2. 클럭 스큐(clock skew) 이벤트 중 발생하는 읽기 왜곡과 데이터 손상
3. 여러 네트워크 분할이 포함된 드문 상황에서 승인된 삽입의 손실

## 해결 현황

"YugaByte DB는 이 세 가지 문제와 여러 다른 문제들을 1.2.0 버전에서 모두 해결했습니다."
