# YugaByte DB 1.3.1

날짜: 2019-09-04

Jepsen은 YugaByte DB 버전 1.3.1의 베타 직렬화 가능 SQL 트랜잭션 지원에 초점을 맞춰 공동 분석을 수행했습니다. 평가 결과 여러 안전성 문제가 확인되었습니다.

## 주요 발견사항

- `DEFAULT` 컬럼 초기화를 `NULL`로 설정하는 경합 조건
- 태블릿 서버가 마스터 노드 연결을 잃을 때 발생하는 직렬화 가능성 위반(G2-item)
- 테이블 생성과 관련된 DDL 문 문제
- 네트워크 분할 시 느린 복구와 프로세스 누수를 포함한 가용성 문제

## 해결 현황

YugaByte는 G2 직렬화 가능성 문제와 일부 추가 버그를 해결했으며, 나머지 문제에 대한 작업을 계속하고 있습니다.
